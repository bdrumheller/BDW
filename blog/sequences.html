Integer Sequences

10/12/2019

Sequence finding problems are often trivial. They follow some basic arithmetic operations from one term to the next. Here, I introduce the concepts of tupular and self-referential sequences. These allow for more interesting patterns to emerge as n-ary operators including min, max, median, cardinality, etc. can now be applied to each term.


Tupular Sequences

Consider a sequence of tuples in the form (a,b,c, ... n) where a,b,c, ... n are integers. Terms in a tupular sequence are generated recursively by applying an n-arity function to one or more previous sequence entries. Try to attempt the examples below before viewing the solutions.


Example 0

Find the next three terms in the sequence:

 (0,2), (5,2), (5,12), (55,12)

Example 1

Find the next three terms in the sequence:

 (7,5,7), (7,7,7), (10,7,7), (10,9,7)


Self-Referential Sequences

Consider an integer sequence of the form a1, a2, a3, a4, ... an where a1, a2, ... an are integers. A self-referential sequence is formed by taking entries from the sequence an with indicies based on values in the sequence. Try to attempt the examples below before viewing the solutions.


Example 0

Find the next three terms in the sequence:

 (0,4,49,289)

Example 1

Find the next three terms in the sequence:

 (0,1,3,13,89)
