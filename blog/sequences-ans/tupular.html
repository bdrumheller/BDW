Tupular Sequences Answers

Example 0

Find the next three terms in the sequence:

 (0,2), (5,2), (5,12), (55,12)


Answer: The next three terms are: (55, 1862), (3265305, 1862), (3265305, 10650060216112).

Explanation: To generate the next term in the sequence, replace the minimum of the term with the square of the difference of each term plus one. The Python code below generates the first 7 terms in the sequence.
x = [0,2] for _ in range(7): print(x) min_index = x[1] == min(x) x[min_index] += (x[1]-x[0])**2+1

Example 1

Find the next three terms in the sequence:

 (7,5,7), (7,7,7), (10,7,7), (10,9,7)


Answer: The next three terms are: (10,9,8), (10,9,8), (10,9,8).

Explanation: To generate the next tuple in the sequence, add the number of 7s to the index corresponding to the cardinality of the tuple. The Python code below generates the first 7 terms in the sequence.
from collections import Counter x = [7,5,7] for _ in range(7): print(x) counter = Counter(x) x[len(set(x))-1] += counter[7]
